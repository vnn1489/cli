---- NGINX
ps -C nginx -f ---> LIST ALL MASTER & WORKER PROCESS
nginx -s reload ---> GRACEFULLY RELOAD WITHOUT STOPPING THE SERVER OR DROPPING ANY ACTIVE CONNECTIONS
nginx -v ---> PROVIDES A QUICK WAY TO CHECK THE NGINX VERSION
nginx -V ---> PROVIDES DETAILED INFORMATION ABOUT THE NGINX INSTALLATION, INCLUDING COMPILE-TIME OPTIONS AND MODULE DETAILS
nginx -t ---> USED FOR SYNTAX CHECKING AND ERROR DETECTION IN YOUR CONFIGURATION FILES
nginx -T ---> USED TO DISPLAY THE COMPILED AND EFFECTIVE CONFIGURATION THAT NGINX WILL USE
nginx -s AllSignal ---> AllSignal (stop, quit, reload, reopen, upgrade)
